# Dicoding - Multiplatform
Ini merupakan dokumentasi hasil pengerjaan project latihan pada learning path multiplatform by dicoding. Pada project ini diberikan study case untuk membangun aplikasi yang memuat informasi tempat wisata. Pada aplikasi ini saya mengangkat informasi tempat wisata di wilayah malang.

# Fitur
  1. Menampilkan gambar dan informasi tempat wisata.

# Teknologi : 
  1. Dart Language
  2. flutter 

# Cara Instalasi 
  1. Clone repository.
  2. flutter pub get (di dalam terminal IDE)
  3. flutter run (untuk menjalankan program)
